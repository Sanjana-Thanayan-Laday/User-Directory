Users Directory

Project Description 
The Users Directory is a frontend web application that displays user details by fetching data from an external API.
The project demonstrates API integration, client-side data handling, and responsive web design using basic web technologies.

Project Explanation
When the application loads, it sends a request to the JSONPlaceholder Users API using JavaScript. The API returns user data in JSON format, 
which is then processed and displayed dynamically on the webpage. The application allows users to search and sort the data without reloading the page.

Features
•	Fetches user data from an external API
•	Displays Name, Email, Company, and City
•	Search users by Name or Email
•	Sort users alphabetically (A–Z / Z–A)
•	Responsive design for mobile and desktop

Technologies Used
•	HTML
•	CSS
•	JavaScript
•	JSONPlaceholder API

API Used
https://jsonplaceholder.typicode.com/users

How to Run the Project
1.	Download or clone the project files
2.	Open the index.html file in any modern web browser
3.	The user data will load automatically from the API

Purpose of the Project
This project was created to demonstrate:
•	API integration
•	JavaScript DOM manipulation
•	Client-side search and sorting
•	Responsive frontend design

Conclusion
The Users Directory project demonstrates a complete frontend workflow, from fetching data to displaying it interactively.
It is simple, efficient, and follows good practices for building small-scale web applications.
